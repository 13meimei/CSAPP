## 6.1 存储技术

SRAM：做高速缓存，利用双稳态存储单元，有电就永保其值，即使有干扰

DRAM：做主存或图形系统的帧缓存，利用电容，对干扰敏感，需要刷新

![](sram_dram.jpg)

闪存：非易失性村粗器  

SSD

磁盘： 访问时间=寻道时间+旋转时间+传送时间

磁盘驱动器为何着逻辑块号和实际物理磁盘扇区之间的映射

**一个典型SSD的性能特征**：顺序读写性能行相当，顺序读比顺序写稍微快；随机写比随机读慢一个数量级
